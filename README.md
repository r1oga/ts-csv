# CSV

Load, parse and filter a csv file.

# Requirements

- [Deno](https://deno.land/)
- [Denon](https://deno.land/x/denon)

# Getting started

In two different terminals:  
`denon -c src/denon.json build`
`denon -c build/denon.json start`
